Defining Data Quality

Quality is not binary. It is multidimensional. According to ISO/IEC 25012, we must measure data against specific characteristics to determine its "fitness for use."

Data Maturity Assessment

Target State Current State

Accuracy

Does the data correctly represent the real-world entity? (e.g., correct address)

Completeness

Is all required data present? (e.g., no missing mandatory fields)

Consistency

Is the data uniform across different systems and datasets?

Timeliness

Is the data up-to-date and available when needed for decision making?
